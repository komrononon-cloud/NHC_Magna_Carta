# Magna Carta: Voices and the Evolution of Democracy

Static website for National History Challenge 2026: Voices.

## Open Locally

Open `index.html` directly in a browser such as Chrome, Safari, or Edge.

## Publish with GitHub Pages

1. Create a public GitHub repository.
2. Upload the site files, including `index.html`, `styles.css`, `script.js`, `.nojekyll`, and the `assets` folder.
3. In repository settings, open Pages.
4. Set the source to deploy from the main branch and the root folder.
5. Save. GitHub will give you a public website URL.

The site is written in plain HTML, CSS, and JavaScript, so it is ready for GitHub Pages, Netlify, Cloudflare Pages, or any other static host.
